Name: Premium Digital Tools

Description: A React-based e-commerce style web app where users can browse premium digital products, add them to a cart, and manage their selections — all with a smooth tabbed
 UI and real-time cart count.

Technologies: React 19, Vite, Tailwind CSS v4, DaisyUI, Lucide React, React Toastify
Features:

🛒 Cart System — Add products to cart and switch between the Products and Cart
 tabs with a live item count
🔔 Toast Notifications — Real-time feedback on user actions via React Toastify
💰 Pricing & Workflow Sections — Dedicated sections for pricing plans and a visual workflow guide to help users get started
